# Madhavan E-Commerce Sales Dashboard | Power BI

##  Project Overview
Analyzed e-commerce sales data for Madhavan store using Power BI to uncover 
business insights including revenue trends, profit by category, customer 
performance, and state-wise sales distribution.

##  Tools Used
- **Power BI** — Interactive dashboard creation and visualization
- **Excel** — Data cleaning and preparation
- **Power Query** — Data transformation

##  Key Business Questions Answered
1. What is the total sales amount and profit?
2. Which state generates the highest revenue?
3. Which product sub-category is most profitable?
4. Who are the top performing customers?
5. Which payment mode is most preferred?
6. How does profit vary across months?

##  Key Insights
- **Total Sales Amount:** 438K
- **Total Quantity Sold:** 6K
- **Total Profit:** 37K
- **Top State:** Maharashtra
- **Most Profitable Sub-Category:** Printers
- **Most Used Payment Mode:** Cash on Delivery (44%)
- **Peak Profit Month:** November
